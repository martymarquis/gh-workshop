# Intermediate GitHub: Practical Publishing Repository Template

> 🛑 If you arrived here first, stop. Follow the instructions here:<br>
> https://github.com/cascadiarconf-gh-workshops/intermediate-practical-publishing

This repository is a starter Quarto website for the workshop. Create your own public repository using **Use this template**, clone your new repository, and follow the workshop instructions to customize and publish the site.

The GitHub Pages workflow is stored in `workflow-examples/`. It will not run until you add a copy to `.github/workflows/publish.yml` during the publishing exercise.

### Preview locally

```shell
quarto preview
```

### Render locally

```shell
quarto render
```

Rendered website files are written to `_site/` and are intentionally excluded from Git.
